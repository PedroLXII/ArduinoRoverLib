This is an AVR Studio project with the core files from the Arduino.cc project incorporated.  It is configured
as a cpp project and has the necessary helper files to eliminate linking and other errors.  The object
files from the output folder can be used to create an archive that the final project will include.

docs - html generated documentation for the source

library - core code from Arduino.cc project

output - project directory for build results

source - Entrance point and contribution source files
